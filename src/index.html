    About Notscape
    N
    Notscape 0.6.0
    The browser that makes every website look like the old internet.
    Tiny user guide
    • Sign On at launch, then type a URL in the Location: bar (or search).
    • Old Internet: ON/OFF (toolbar) toggles the whole retro look.
    • ★ Mods — presets, sliders (Vintage, GeoCities-ify) and look toggles.
    • ✨ Flourishes — starfields, snow, sparkle cursors & more, remembered per site.
    • ➕ Add (bookmarks bar) saves the current page; bookmarks also appear on your start page.
    • The start page is an RSS reader — add channels in the right-hand panel.
    • Edit → Preferences — screen name, ad-blocking, Safe Mode, clear data.
    • Keys: F5 reload · Ctrl+L address bar · F12 devtools · Esc close.

    Made with <3 and too many <table> tags. MIT licensed.
