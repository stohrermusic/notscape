# Notscape sound assets

Notscape will play these files if they exist; otherwise it falls back to your
system's text-to-speech voice saying the line.

| File                  | Plays when…                          | Spoken fallback        |
|-----------------------|--------------------------------------|------------------------|
| `welcome.wav`         | you sign on / connect                | "Welcome!"             |
| `youve-got-mail.wav`  | you navigate to Gmail                 | "You've got mail!"     |

Drop in your own clips with those exact names to use the real audio. `.wav`,
`.mp3`, and `.ogg` all work if you keep the `.wav` filename, but `.wav` is
recommended. These are not bundled because the originals are copyrighted.
